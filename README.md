# GalbaGrama*

Suite de pasatiempos y rompecabezas de palabras y lógica de GalbaHub.

## Estructura del repositorio

Este repositorio está organizado como una suite de juegos independientes:

- **[Nonograma](file:///home/ggalba/Escritorio/Proyectos/apps/Galbagrama/Nonograma):** Descubre las imágenes ocultas resolviendo cuadrículas lógicas (Estética retro Game Boy Color con sonidos de 8 bits y tres modos de juego: Normal, Montaña y Flash).

## Requisitos de ejecución local

Para iniciar cualquiera de los juegos de forma independiente, accede a su respectiva carpeta y ejecuta:

```bash
npm install
npm run dev
```
